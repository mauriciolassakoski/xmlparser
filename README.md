# File Parser

This project contains examples of reading files in various formats such as xml, xlsx, json ..


## Requirements:

 * Java 8 installed;
 * Maven installed


## Installation

  * Clone the project from github
  `https://github.com/mlassakoski/fileparser.git`

#### Download project dependencies

Maven dependencies
  * cd fileparser;
  * mvn clean install;
